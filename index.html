Event Meetup App

React, Node, Gherkin, Google Calendar API

Event Meetup App is a serverless, progressive web application (PWA) with React using the Google Calendar API to fetch upcoming events.

Users can search for a city and view a list of events hosted in the selected city. The app also allows a user to view charts that display how many events will take place on certain days as well view the popularity of the event.

Tech Specs: React, Node, Gherkin, Google Calendar API

Chatterbox

React Native, Gifted-Chat, Firebase

Chatterbox is an app for mobile devices built with React Native and Expo, chat interface styled with the gifted-chat library, and stores data with Google's Firebase.

Users are able to share images (take a picture or choose from media library) once the user grants access to their local media library and camera. Location sharing is enabled by the user granting the app permission to read user location data.

Tech Specs: React Native, Gifted-Chat, Firebase
